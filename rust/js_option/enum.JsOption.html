Enum js_option::JsOption

source · [−]
pub enum JsOption<T> {
    Some(T),
    Null,
    Undefined,
}
Expand description

An Option-like type with two data-less variants in addition to Some: Null and Undefined.

Variants

Some(T)

Some value T

Null

Explicit absence of a value

Undefined

Implicit absence of a value
